Dnn Icons

The official repository for the DNN Icons Font

View on GitHub Download .zip Download .tar.gz

DNN contains hundreds of different icons used throughout the platform. The DNN Icons project contains commonly used DNN icons in an easy to use format which also helps improve overall page speed. If you have ever used a icon font like FontAwesome or GlyphIcons, then you already know how to use DNNIcons.
